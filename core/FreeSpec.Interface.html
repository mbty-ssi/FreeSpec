Regural Interfaces

We consider an Interface to be a set of operations whose result type only is known. This is usually encoded thanks to a parameterized type in functional programming language, where the type parameter is the type of the result.

Labeled Interfaces

We can label an interface with a particular type to enrich a primitive call with a context. The main idea is this label will be stripped before extraction.

Inductive LabeledInterface
          (I: Interface)
          (L: Type)
  : Interface :=
| labeled_instruction (A: Type)
                      (i: I A)
                      (l: L)
  : LabeledInterface I L A.

Arguments labeled_instruction [I L A] (i l).
